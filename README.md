# Phishing Email Detection & Awareness System

## Overview
This project demonstrates phishing email analysis and awareness reporting using phishing indicators, email header analysis, and URL investigation techniques.

The objective is to identify phishing attacks, classify email threats, and create employee awareness guidelines to reduce cyber risks in organizations.

---

## Features
- Phishing email analysis
- Email header inspection
- Fake domain detection
- Suspicious URL investigation
- Risk classification
- Employee awareness recommendations

---

## Tools Used
- MXToolbox Email Header Analyzer

- Browser Developer Tools
